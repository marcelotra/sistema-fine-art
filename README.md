# Sistema de Orçamento Fine Art

Este é um projeto React independente para orçamentos de impressão Fine Art.

## Como Rodar

1.  Abra o terminal nesta pasta: `c:/Users/Pichau/Documents/sistema-fine-art`
2.  Instale as dependências (se ainda não fez):
    ```bash
    npm install
    ```
3.  Rode o projeto:
    ```bash
    npm run dev
    ```
4.  Acesse `http://localhost:5173` no navegador.

## Estrutura

- `src/screens/QuoteScreen.tsx`: Tela principal com a lógica de orçamento.
- `src/App.tsx`: Ponto de entrada.
- `tailwind.config.js`: Configuração de estilos.
